Descripción del Dataset

Fuente y contexto

El dataset utilizado es el registro histórico de Comercio Exterior de México por País (1993–2025), disponible en datamx.io y construido a partir de registros oficiales del Banco de México.

Se enriqueció con el tipo de cambio MXN/USD mensual obtenido de la API del Banco de México (Serie SF43718).

Nota

Licencia: Creative Commons CCZero (dominio público). Los datos son de fuente verificable y abierta.

Dimensiones

Atributo Detalle
Registros 184,392 filas
Columnas originales 8
Columnas tras enriquecimiento 12
Cobertura temporal Enero 1993 – 2025 (mensual)
Países socios 180 países en 6 continentes

Variables

Variables originales

Variable Tipo Descripción
PERIODO Fecha Mes y año del registro
CONTINENTE Categórica Continente del país socio
ID_CONTINENTE Numérica Código numérico del continente
PAIS Categórica País socio comercial
ID_PAIS Numérica Código numérico del país
FLUJO Categórica Exportaciones o Importaciones
ID_FLUJO Numérica Código del flujo (1 o 2)
VALOR_USD Numérica Valor del intercambio en USD

Variables derivadas

Variable Tipo Descripción
ANIO Numérica Extraída de PERIODO
MES Numérica Extraída de PERIODO
TIPO_CAMBIO Numérica MXN/USD mensual (Banxico SF43718)
VALOR_MXN Numérica VALOR_USD × TIPO_CAMBIO
BALANCE Numérica Exportaciones − Importaciones por país-mes
SUPERAVIT Binaria 1 = superávit, 0 = déficit (variable objetivo)
CONTINENTE_ENC Numérica LabelEncoder sobre CONTINENTE
PAIS_ENC Numérica LabelEncoder sobre PAIS

Pertinencia del dataset

El dataset fue elegido por cuatro razones concretas:

  1. Escala suficiente: 184,392 registros mensuales permiten entrenamiento robusto con división 80/20.
  2. Variables relevantes: 8 variables originales más variables derivadas que cubren aspectos económicos, geográficos y temporales.
  3. Compatibilidad con ambas tareas: el mismo dataset alimenta tanto el modelado supervisado como el clustering.
  4. Fuente verificable: licencia CCZero y origen en registros oficiales del Banco de México y la Secretaría de Economía.

Limitaciones conocidas

Advertencia

Valores nominales: VALOR_USD no está deflactado por inflación. Parte del crecimiento observado en la serie de tiempo refleja inflación acumulada y no necesariamente mayor volumen real de comercio. Como trabajo futuro se recomienda deflactar con el Índice Nacional de Precios al Productor (INPP).

  • Granularidad: datos agregados a nivel país-mes; no permite análisis por producto o fracción arancelaria.
  • Año incompleto: 2025 cubre solo los meses disponibles a la fecha de extracción.
  • ID_CONTINENTE: presenta 28,368 valores faltantes (15.4%), por lo que fue excluida del modelado. La columna CONTINENTE contiene la misma información sin faltantes.

Proceso de enriquecimiento

import requests
import pandas as pd

# Descarga del tipo de cambio MXN/USD desde la API de Banxico (Serie SF43718)
url = (
    "https://www.banxico.org.mx/SieAPIRest/service/v1/series/"
    "SF43718/datos/1993-01-01/2025-12-31"
)
headers = {"Bmx-Token": "<token>"}
response = requests.get(url, headers=headers)
datos_tc = response.json()["bmx"]["series"][0]["datos"]

df_tc = pd.DataFrame(datos_tc)
df_tc["fecha"] = pd.to_datetime(df_tc["fecha"], format="%d/%m/%Y")
df_tc["TIPO_CAMBIO"] = df_tc["dato"].astype(float)

# Unión con el dataset principal por año y mes
df = df.merge(df_tc[["fecha", "TIPO_CAMBIO"]], on="fecha", how="left")
df["VALOR_MXN"] = df["VALOR_USD"] * df["TIPO_CAMBIO"]
